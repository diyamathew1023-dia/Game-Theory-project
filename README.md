# Game Theory Project

This project explores the fundamental concepts of Game Theory, including strategies, payoff matrix, saddle point, and different types of games. It provides a mathematical approach to analyze decision-making in competitive situations.

## Topics Covered
- Pure and Mixed Strategies
- Saddle Point
- Minimax Theorem
- Types of Games

## Tools Used
- Mathematics
- Analytical Methods
